## ✨ Características Principales

- 🗳️ **Sistema de Votación en Real-Time:** Sincronización instantánea con Supabase.
- 🎮 **Evento Final "Asriel":** Combate interactivo inspirado en RPGs clásicos.
- 🎥 **Live Dashboard:** Control total de la ceremonia para administradores.
- 📱 **Diseño Ultra-Responsivo:** Optimizado para móviles y pantallas gigantes.
- ⚡ **Optimización de Fuentes:** Uso de `next/font` para Geist y Determination Mono.
